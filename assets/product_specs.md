# Product Specifications & Rules

1. **Discount Codes**:
   - The code `SAVE20` grants a 20% discount on the total price.
   - The code `WELCOME50` grants a $50 flat discount.
   - Any invalid code must display "Invalid Code" in red text.

2. **Shipping Rules**:
   - Standard shipping is free ($0).
   - Express shipping costs $20 extra.

3. **Form Validation**:
   - Email field is mandatory.
   - If email format is incorrect, show "Invalid Email" in the `#email_error` span.
